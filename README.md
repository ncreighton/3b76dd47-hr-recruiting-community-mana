# HR & Recruiting Community Manager Discord Bot

Automated moderation, welcome flows, role management, and engagement analytics for HR & Recruiting communities. Purpose-built for HR & Recruiting community needs.

## Setup
1. Set `TELEGRAM_BOT_TOKEN` in `.env`
2. `pip install -r requirements.txt`
3. `python main.py`
